Getting Started
---------------

Dymion is a professional physics library for kinematics and dynamics.

.. code-block:: bash

   pip install dymion

Features
--------

* **Vector Engine:** 2D and 3D vector operations.
* **Kinematics:** Linear and circular motion.
* **Dynamics:** Masses, Forces, Gravity, and Friction.
* **Energy:** Work, Power, and Kinetic/Potential energy.
